# Century 21 Smart CRM - 기능명세서

## 1. 대시보드 (AgentDashboard)

### 1.1 기능 개요
에이전트의 일일 업무 현황을 한눈에 파악할 수 있는 메인 화면

### 1.2 세부 기능

| 기능 | 설명 | 구현 파일 |
|------|------|----------|
| To-do 리스트 | 오늘의 할 일 목록 표시, 완료 체크, 신규 추가 | `views/AgentDashboard.tsx` |
| Contact 현황 | 최근 연락한 고객 및 예정 고객 표시 | `views/AgentDashboard.tsx` |
| 후속 관리 알림 | Contact 기한이 도래/초과한 고객 목록 | `views/AgentDashboard.tsx` |
| 활동 통계 요약 | 신규 등록, Contact 기록 등 주요 지표 | `views/AgentDashboard.tsx` |
| 미니 캘린더 | 오늘 날짜 및 일정 있는 날짜 표시 | `views/AgentDashboard.tsx` |

### 1.3 데이터 소스
- `AppContext`: todos, customers, scheduleEvents, contactRecords

## 2. 고객 DB 관리 (CustomerDB)

### 2.1 기능 개요
등록된 고객 목록 조회, 검색, 필터링 및 일괄 관리

### 2.2 세부 기능

| 기능 | 설명 | 구현 파일 |
|------|------|----------|
| 고객 목록 조회 | 페이지네이션 적용된 고객 테이블 | `views/CustomerDB.tsx` |
| 검색/필터 | 이름, 연락처, 등록일, 그룹별 필터링 | `views/CustomerDB.tsx` |
| 체크박스 선택 | 다중 선택 및 전체 선택 | `views/CustomerDB.tsx` |
| 일괄 삭제 | 선택한 고객 일괄 삭제 | `views/CustomerDB.tsx` |
| 일괄 미루기 | 선택한 고객 Contact 기한 7일 연장 | `views/CustomerDB.tsx` |
| Contact 기록 | 빠른 연락 기록 추가 (모달) | `components/ContactRecordModal.tsx` |
| **Excel 업로드** | CSV 파일로 고객 대량 등록 | `components/ExcelUploadModal.tsx` |
| **Excel 다운로드** | 고객 목록 CSV 추출 | `views/CustomerDB.tsx` |
| **그룹 관리** | 고객 그룹 생성/수정/삭제 | `components/GroupManageModal.tsx` |

### 2.3 데이터 구조
```typescript
interface Customer {
  id: string;
  name: string;
  phone: string;
  email?: string;
  address?: string;
  memo?: string;
  photo?: string;           // Base64 이미지
  groups?: string[];        // 그룹 ID 배열
  attachments?: CustomerAttachment[];
  lastContact: string;
  registrationDate: string;
  status: 'ACTIVE' | 'INACTIVE';
  tier?: 'Platinum Elite' | 'Gold Medal' | 'Silver Link';
  contactDueDays: number;
  isNew?: boolean;
}
```
