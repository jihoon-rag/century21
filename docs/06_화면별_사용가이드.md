# Century 21 Smart CRM - 화면별 사용 가이드

## 1. 대시보드

### 1.1 화면 구성
```
┌─────────────────────────────────────────────────────────┐
│  To-do List          │  Contact 현황                    │
│  ☐ 매물 홍보 전단지  │  [Contacts] [To-do]             │
│  ☐ 강남역 부지 확인  │  ┌─────────────────────────────┐ │
│  + 새로운 할 일      │  │ 홍길동 | 오늘 | 010-1234   │ │
├──────────────────────┤  │ 김중앙 | +3일 | 010-5678   │ │
│  미니 캘린더         │  └─────────────────────────────┘ │
│  [< 2024년 5월 >]    │                                  │
│  일 월 화 수 목 금 토│  후속 관리 필요 고객             │
│        1  2  3  4  5 │  ⚠ 이송파 (-3일)                │
│  ...                 │  ⚠ 정마포 (-7일)                │
└─────────────────────────────────────────────────────────┘
```

### 1.2 주요 기능

| 영역 | 조작 | 결과 |
|------|------|------|
| To-do 체크박스 | 클릭 | 완료 처리 |
| + 새로운 할 일 | 입력 후 엔터 | 할 일 추가 |
| Contact 탭 | 탭 클릭 | Contacts/To-do 전환 |
| 고객 카드 | 클릭 | 고객 상세 이동 |
| 기록 버튼 | 클릭 | Contact 기록 모달 |
| 캘린더 날짜 | 클릭 | 일정 관리 이동 |

## 2. 고객 DB

### 2.1 화면 구성
```
┌─────────────────────────────────────────────────────────┐
│  고객 DB 관리                    [업로드][다운로드][그룹][등록] │
├─────────────────────────────────────────────────────────┤
│  필터: [고객명    ] [연락처    ] [그룹 ▼] [날짜 ~ 날짜]    │
│        [필터 초기화]                              [검색]    │
├─────────────────────────────────────────────────────────┤
│  총 10명의 고객                               2명 선택됨   │
├─────────────────────────────────────────────────────────┤
│  ☐ │ Contact │ 고객명      │ 연락처       │ 최근    │관리│
│────┼─────────┼─────────────┼──────────────┼─────────┼────│
│  ☑ │ +3일    │ 홍길동 NEW  │ 010-1234... │ 05.15  │기록│
│  ☐ │ Today   │ 삼성동건물주│ 010-2345... │ 05.31  │기록│
│  ☑ │ -1일    │ 홍길준      │ 010-3456... │ 04.14  │기록│
├─────────────────────────────────────────────────────────┤
│  [삭제(2)] [미루기(7일)]              [< 1 2 3 4 5 >]   │
└─────────────────────────────────────────────────────────┘
```

### 2.2 주요 기능

| 버튼 | 기능 | 설명 |
|------|------|------|
| 업로드 | Excel 업로드 | CSV 파일로 대량 등록 |
| 다운로드 | Excel 다운로드 | 필터링된 고객 CSV 추출 |
| 그룹 | 그룹 관리 | 그룹 생성/수정/삭제 |
| 등록 | 신규 등록 | 고객 상세 페이지 이동 |
| 기록 | Contact 기록 | 빠른 연락 기록 추가 |
| 삭제 | 일괄 삭제 | 선택한 고객 삭제 |
| 미루기 | 기한 연장 | Contact 기한 7일 연장 |

### 2.3 필터 사용법
1. 고객명/연락처 입력 후 [검색] 클릭
2. 그룹 선택 시 해당 그룹 고객만 표시
3. 날짜 범위 지정으로 등록일 필터링
4. [필터 초기화]로 모든 필터 해제

│  ┌──────────────────────────────────────────────────┐   │

### 3.2 주요 기능

| 영역 | 조작 | 결과 |
|------|------|------|

| 그룹 버튼 | 클릭 | 그룹 토글 (복수 선택 가능) |
| 파일 추가 | 클릭 | 첨부 파일 선택 |
| 파일 X | 클릭 | 첨부 파일 삭제 |
| 수정하기 | 클릭 | 정보 저장 |

## 7. 공통 UI 요소

### 7.1 상단 네비게이션 (PC)
```
┌─────────────────────────────────────────────────────────┐
│  🏠 Century 21 CRM          🔔(2) ⚙️ 설정   [이동현] 👤 │
└─────────────────────────────────────────────────────────┘
```

- 🔔 알림: 클릭 시 알림 패널
- ⚙️ 설정: 클릭 시 설정 모달
- 프로필: 사용자 정보
