## 📌 MVP 현황 요약

### ✅ 구현 완료 (UI + 프론트엔드 로직)
- 고객 등록/수정/삭제/조회
- Excel 대량 업로드/다운로드
- 고객 그룹화 관리
- 프로필 사진/파일 첨부
- 일정 관리 (캘린더)
- 통계 대시보드
- 목표 설정 및 달성률
- SMS/이메일 발송 UI
- AI 비서 채팅 UI
- 반응형 디자인 (PC/Mobile)

### ⚠️ 백엔드 연동 필요
- 사용자 인증 (로그인)
- 데이터 영구 저장
- 실제 SMS/이메일 발송
- 푸시 알림 서비스
- AI API 연동

## 🛠 기술 스택

```
Frontend: React 19 + TypeScript + Vite
Styling: Tailwind CSS
Charts: Recharts
State: React Context API
Hosting: Vercel
```
